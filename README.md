## Pourquoi ce repos?

Je condense peu à peu dans ce repository des petits exercices/katas que je peux faire dans differentes techno.
L'avancée n'est ni régulière ni homogene, bien qu'il y a eu un debut de php ou de typescript pour le moment c'est le code c# qui est le plus complet.

Vous pourrez y trouver aussi bien des resolutions de kata, que le debut de la mise ne oeuvre d'un mini moteur rpg en clean archi.

Le but de ce repository est la mise en avant des bonnes pratiques de developpement comme TDD, d'archi logiciel comme clean archi, ou encore un peu de devops si l'occasion se presente.
